# Frontend Intern Tasks

Welcome to the SSC Frontend Internship Repository.

This repository contains:
- React tasks
- Next.js assignments
- UI challenges
- Tailwind CSS practice
- Responsive design exercises

## Tech Stack
- React.js
- Next.js
- Tailwind CSS
- JavaScript

## Workflow
1. Clone repository
2. Create new branch
3. Complete assigned task
4. Commit changes
5. Push branch
6. Create Pull Request

## Branch Naming
feature/task-name

Example:
feature/navbar-ui

## Important Rules
- Do not push directly to main
- Write clean code
- Make UI responsive
- Use reusable components
